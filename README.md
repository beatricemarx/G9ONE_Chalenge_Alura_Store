Análise de Dados - AluraStore Brasil 🛒
Este projeto realiza uma análise exploratória de dados (EDA) e o tratamento de múltiplos datasets de vendas da AluraStore Brasil. O objetivo é consolidar informações de diferentes fontes para identificar padrões de consumo, avaliações de clientes e performance geográfica das vendas.

📋 Sobre o Projeto
O notebook aborda desde a importação de dados brutos armazenados em CSVs externos até a estruturação de um dataset unificado para análise. A análise foca em entender o comportamento do e-commerce através de diversas métricas de negócio.

Estrutura dos Dados

O dataset consolidado contém as seguintes informações:

Produto e Categoria: Identificação dos itens (ex: eletrônicos, móveis, brinquedos).

Preço e Frete: Valores financeiros das transações e custos logísticos.

Data e Local: Registro temporal e geográfico (UF) das compras.

Avaliação: Feedback dos clientes (escala de 1 a 5).

Pagamento: Métodos utilizados e quantidade de parcelas.

Geolocalização: Coordenadas (latitude e longitude) das transações.
